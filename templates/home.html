Technical Methodology
1. Recommendation Engine (Cosine Similarity)

The system utilizes a vector-space model to represent movies as high-dimensional vectors. Similarity is calculated using the dot product of normalized vectors:

Similarity(A, B) = (A · B) / (||A|| * ||B||)

This represents the cosine of the angle between two movie vectors. A value of 1 indicates perfect alignment.

2. Sentiment Analysis (NLP Pipeline)

Reviews are processed through a TF-IDF (Term Frequency-Inverse Document Frequency) vectorizer and classified using a Logistic Regression model trained on 50,000 global review transcripts.

3. Data Engineering

The system employs a multi-source pipeline (TMDB API & IMDB Scraping) to ensure data robustness and real-time metadata accuracy.
